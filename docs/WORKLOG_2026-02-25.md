## Operational Notes Learned
- Cloud Run service is reachable 24/7 by URL, but compute does not run continuously when `min-instances=0`.
- Cost is not only Cloud Run compute; major variable cost is API usage:
  - Vision OCR
  - Gemini generation
  - Vertex search (if enabled)
- If no scans occur, costs are typically very low (near-zero), with only small residual platform/storage logging costs possible.
